package: '@nature-ui/close-button'
description:
  CloseButton is essentially a button with a close icon. It is used to handle
  the close functionality in feedback and overlay components like Alerts,
  Toasts, Drawers and Modals
---

## Import

```js
import { CloseButton } from '@nature-ui/core';
```

## Usage

```jsx
<CloseButton />
```

### Button Size

Pass the `size` prop to adjust the size of the close button. Values can be `sm`,
`md` or `lg`.

```jsx
<Stack row spacing='1rem'>
	<CloseButton size='sm' />
	<CloseButton size='md' />
	<CloseButton size='lg' />
</Stack>
```

## Props

The CloseButton composes `Box` component.

| Name         | Type             | Default | Description                              |
| ------------ | ---------------- | ------- | ---------------------------------------- |
| `aria-label` | `string`         |         | An accessible label for the close button |
| `isDisabled` | `boolean`        |         | If `true`, the button will be disabled   |
| `color`      | `string`         |         | The color of the close icon              |
| `size`       | `sm`, `md`, `lg` | `md`    | The size of the close button             |
